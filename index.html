Three Department Enterprise Network Design | Data Communication & Network

Designed and simulated a three-department enterprise network (HR, Finance, and IT) for XYZ Corporation using Huawei eNSP, based on a multi-site architecture across Beijing HQ and branch offices in Shanghai, Chengdu, and Kuala Lumpur. The network addressed real-world enterprise issues such as lack of segmentation, inefficient routing, and centralized service management.

Implemented key networking services across multiple layers, including DHCP for centralized IP management, DNS for name resolution, and HTTP/HTTPS for web service accessibility. Designed VLAN segmentation and subnetting to isolate departments and improve security, while configuring OSPF with authentication to enable secure and efficient inter-branch routing.

Simulated transport-layer behavior using TCP-based services to demonstrate reliable communication and analysed connectivity using tools such as ping and traceroute. Performed systematic testing and troubleshooting to validate network performance and identify issues related to routing, IP configuration, and service accessibility.

Perak AirWatch – IoT Aircraft Monitoring System | Internet of Things

Designed and developed an IoT-based aircraft monitoring system to track real-time flight activity over Perak, Malaysia using the OpenSky Network API. The system continuously collected aircraft telemetry data including latitude, longitude, altitude, and velocity at 2 minute intervals over a 7 day period, enabling reliable analysis of air traffic patterns.

Built a layered system architecture integrating data collection, processing, storage, and visualization components. Implemented a Python-based data collector to fetch and filter flight data within defined geographical boundaries, with persistent storage using CSV files for lightweight and cost-efficient data management.

Developed a web-based dashboard using Flask, Leaflet.js, and Chart.js to visualize real-time aircraft positions and historical trends, including flight frequency and altitude distribution. Integrated WebSocket communication (Flask-SocketIO) to enable live data updates without page refresh, enhancing interactivity and user experience.

Conducted data analysis to identify traffic patterns, peak flight periods, and common cruising altitudes (9,000–11,000 meters), demonstrating practical insights into regional airspace activity. The system was implemented using fully open-source tools with zero operational cost, showcasing an efficient and scalable IoT solution.
